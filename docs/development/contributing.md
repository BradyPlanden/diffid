!!! info "Coming Soon"

## Quick Start

```bash
# Fork and clone
git clone https://github.com/YOUR_USERNAME/diffid.git
cd diffid

# Set up environment
uv sync
uv run maturin develop

# Run tests
uv run pytest -v
cargo test

# Make changes and submit PR
```

## Areas to Contribute

- Bug fixes
- New algorithms
- Documentation
- Examples
- Tests
